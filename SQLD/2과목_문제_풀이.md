3. TCL
- 논리적인 작업의 단위를 묶어 DML에 의해 조작된 결과를 작업 단위별로 제어하는 명령어인 COMMIT, ROLLBACK, SAVEPOINT 등이 여기에 해당하며, 일부에서는 DCL로 분류하기도 한다

4. 절차적 데이터 조작어
- 어떻게 데이터를 접근해야 하는지 명시한다
- **비절차적 데이터 조작어**는 사용자가 무슨 데이터를 원하는지만 명세

5. SQL

- DCL: GRANT, REVOKE
- TCL: COMMIT, ROLLBACK
