# Comprehension
- 사용되는 방법, 문법은 동일하지만 어떤 자료형에 적용시키는지에 따라 다름
- 조건문, 반복문 등의 개념을 알고 있어야 컴프리헨션 사용가능함

</br>

## 리스트 컴프리헨션
- 기존의 1부터 10까지 정수를 순서대로 가지고 있는 리스트
```python
num = []
for i in range(1, 11):
  num.append(i)
```

</br>

- 컴프리헨션으로 표현
```python
[i for i in range(10)]
```
