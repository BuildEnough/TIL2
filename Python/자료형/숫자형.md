# 숫자형
- 숫자 형태로 이루어진 자료형
- 123 같은 정수
- 12.34 같은 실수
- 8진수와 16진수


## 정수형
- 정수를 뜻하는 자료형
```
a = 123
a = -178
a = 0
```

## 실수형
- 소수점이 포함된 숫자
```
a = 1.2
a = -3.45
```
- 컴퓨터식 지수 표현 방식
```
a = 4.24E10  => 4.24*10**10
a = 4.24e-10  => 4.24*10**-10
```

## 8진수
- 0o 또는 0O 시작
```
a = 0o177
```

## 16진수
- 0x 시작
```
a = 0x8ff
b = 0xABC
```
