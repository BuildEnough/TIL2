# 숫자형 사칙연산
## 사칙연산
- `+` `-` `*` `/`
```
>>> a = 3
>>> b = 5
>>> a + b
8   
>>> a * b
15
>>> a / b
0.6
>>> a - b
-2
```

## x의 y제곱을 나타내는 `**`연산자
- `x ** y`처럼 사용했을때 x의 y제곱 값을 돌려줌
```
>>> a = 2
>>> b = 3
>>> a ** b
8
```

## 나머지 반환 `%` 연산자
- `%`는 나눗셈의 나머지 값을 돌려줌
```
>>> 5 % 2
1
>>> 7 % 3
1
>>> 10 % 8
2
```

## 몫을 반환  `//` 연산자
- `//` 나눗셈의 몫을 반환해줌
```
>>> 7 // 4
1
```
