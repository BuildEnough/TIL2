# 문자열 연산
## 문자열 더해서 연결(Concatenation)
```
>>> head = 'Python'
>>> tail = ' is fun' 
>>> head + tail
'Python is fun'
```
- 큰 따옴표와 작은 따옴표 섞어써도 상관 없음
```
>>> head = "Python" 
>>> tail = ' is fun' 
>>> head + tail     
'Python is fun'
```

## 문자열 곱하기
- 문자열에서 `*`의 의미는 숫자 곱하기와 다름
- `*`는 문자열의 반복을 뜻함
```
>>> a = 'Pyhton'
>>> a * 2
'PyhtonPyhton'
```
